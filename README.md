# Snepulator

Snepulator is a multi-system emulator for Linux, developed since 2017.

The following systems are supported:

* Sega SG-1000
* Sega Master System
* Sega Game Gear
* ColecoVision

* [gl3w](https://github.com/skaslev/gl3w)
* [libspng](https://github.com/randy408/libspng)
